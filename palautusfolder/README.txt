How to Run Locally on Redmi (Android)
Install the free app "Simple HTTP Server" from Play Store, then:
1. Open the app
2. Tap the folder icon → navigate to your palautus/ folder
3. Tap Start
4. Open Chrome on your Redmi
5. Go to: http://localhost:8080
Or use VS Code + Live Server on a PC and access via your Redmi's browser using your PC's local IP:


http://192.168.x.x:5500
